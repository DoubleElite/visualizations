Heap Visualization Help

  • Click 'Insert Random' to add a random valueto the heap.
  • Click 'Remove Top' to remove the top of the heap.
  • Click 'Clear' to remove all items from heap.
  • Type in the text box and click 'Insert' to add your own value to the heap.
  • Use the radio buttons to toggle between max and min heaps.
Use the control panel to change node background and foreground colors.
Zoom slider is not fully supported by all browsers but you can type a zoome value.
Most browsers will allow you to right-click the canvas area and view the graph as an image so it can be saved.
