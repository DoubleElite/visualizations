Binary Tree Visualization Help

  • Click an existing node to highlight it
    • Click the same node again to delete it and its children
    • Simply start typing to change the label for the highlighted node. Pressing enter or clicking a blank spot on the canvas ends the label creation.
  • Click a node and drag to the lower left to add a left child to the node
  • Click a node and drag to the lower right to add a right child to the node
Use the control panel to change node background and foreground colors.
Zoom slider is not fully supported by all browsers but you can type a zoom value.
Most browsers will allow you to right-click the canvas area and view the graph as an image so it can be saved.
Clicking Postorder, Preorder, or Inorder buttons will show the order of nodes in the corresponding traversal in an alert popup.
